# DigitalBook — АКТ Цифрлық кітап

«Ақпараттық-коммуникациялық технологиялар» пәні бойынша электрондық оқу-әдістемелік кешені.

## Мазмұны

- **15 дәріс** — лекциялық материалдар, бақылау сұрақтарымен
- **13 зертханалық жұмыс** — практикалық тапсырмалармен
- **14 интерактивті тест** — 68 сұрақ, автоматты тексерумен
- **268 сурет** — скриншоттар, диаграммалар, иллюстрациялар

## Іске қосу

```bash
cd app
python3 -m http.server 8080
# Браузерде: http://localhost:8080
```

Немесе кез келген веб-сервер арқылы `app/` бумасын ашыңыз.

## Мүмкіндіктер

- Қазақ тіліндегі толық интерфейс
- Қараңғы/жарық тақырып
- Қаріп өлшемін реттеу
- Кітап бойынша іздеу (Ctrl+K)
- Басып шығару
- Пернетақта навигациясы (Ctrl+←/→)
- Мобильді құрылғыларға бейімделген

## Құрылымы

```
app/
  index.html          — қосымшаның кіру нүктесі
  content.json        — барлық оқу материалдары (JSON)
  css/style.css       — стильдер
  js/app.js           — негізгі қосымша логикасы
  js/test-engine.js   — интерактивті тест жүйесі
  images/             — суреттер (268 файл)
extract_all.py        — .docx → JSON түрлендіргіш
```
